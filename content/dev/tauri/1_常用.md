## 修复WebView空白页

在`tauri.config.json`中

```json
{
  "app": {
    "windows": [
      {
        "visible": false  // 设置不可见
      }
    ],
  }
}
```

在`capabilities/default.json`中

```json
{
  "permissions": [
    "core:window:allow-show"  // 允许JS/TS控制显示窗口
  ]
}
```

在初始化页面，以Vue为例

```ts
import { getCurrentWindow } from '@tauri-apps/api/window';
onMounted(async ()=>{
  const appWindow = getCurrentWindow()
  appWindow.show();
})
```
